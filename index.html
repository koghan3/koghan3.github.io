Classified Capabilities

Full Enforcement Suite

Every tool you need to keep your attention where it belongs.

⏱
[F-01] Core Engine
Pomodoro Timer

Structured focus sessions with short breaks and a long break after every cycle. Fully configurable: work length, break length, sessions per cycle — all under your control.

🚫
[F-02] Site Control
Blacklist & Block

Add any domain to the blacklist. Blacklisted sites are blocked during focus sessions and can be held to a daily time limit. Blocked pages get a proper rejection notice.

✅
[F-03] Exemptions
Whitelist

Exempt trusted domains — work tools, documentation, anything that should never be auto-blocked. Your approved resources are never touched.

📊
[F-04] Intelligence
Activity Records

Every domain you visit is timed and logged. The daily record shows exactly where your hours go. Distractions thrive in the dark — this extension keeps the lights on.

🤖
[F-05] Automation
Auto-Block

Set a daily time threshold. Once a domain exceeds it, it's automatically blacklisted without requiring manual action. Enforcement that doesn't wait for you to notice.

🔒
[F-06] Security
Settings Lock

Lock all settings, blacklists, and whitelists against modification. Unlocking requires a written justification and a waiting period — making self-sabotage inconvenient enough to think twice about.

✋
[F-07] Accountability
Focus Check-Ins

The extension periodically checks in during sessions to verify you're actually working. If you don't respond, the system assumes distraction and acts accordingly.

☕
[F-08] Relief Valve
Break Controls

Request an early break with a stated reason. Extend breaks using a daily bonus allowance. Strict — but not unreasonable.

⏳
[F-09] Limits
Visit Cooldown

Set per-visit time limits on blacklisted domains. When the limit is reached, the site is blocked for a cooldown period before access is restored.

Deploy In Four Steps

Up and running within minutes of installation.

01
Install

02

03
Work

Start a Pomodoro session. Blocked sites stay blocked. Check-ins keep you honest. The timer keeps you moving.

04
Lock

When willpower needs backup, lock your settings. Undoing requires a written reason and a waiting period.

Sideloading Guide

Four Steps to Deploy

01
Extract

Unzip the downloaded file to a permanent folder. Chrome needs the folder to stay in place — don't move or delete it after installing.

02
Extensions Page

Open Chrome and navigate to chrome://extensions in the address bar.

03
Developer Mode

Toggle Developer mode on using the switch in the top-right corner of the extensions page.

04
Load Unpacked

Click Load unpacked and select the folder you extracted in step 1. Done.

[!]  Chrome will show a "developer mode extensions" warning banner — this is normal and harmless. It may also prompt you to disable developer mode extensions on startup; dismiss it and continue. The extension will be submitted to the Chrome Web Store in due course.
