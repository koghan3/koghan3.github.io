LIMITED USE POLICY

The Authoritarian — Chrome Extension
Effective date: 2026-04-01  ·  Last updated: 2026-04-01
AFFIRMATIVE STATEMENT — LIMITED USE COMPLIANCE

The Authoritarian's use of data obtained from Chrome APIs — including browsing activity — adheres strictly to the Chrome Web Store User Data Policy and its Limited Use requirements. Data is used only to provide and improve the extension's core productivity features. It is not transferred, sold, or used for any secondary purpose, including advertising or user profiling.

What the Limited Use Rules Require

The Chrome Web Store Limited Use Policy prohibits extensions from using user data in ways beyond what is necessary to provide the core product. Specifically, data may not be:

  • Transferred to third parties (except as necessary to provide the product, or with explicit user consent)
  • Used or transferred for purposes unrelated to the extension's core functionality
  • Used or transferred for advertising purposes
  • Used or transferred to determine creditworthiness or for lending purposes

How The Authoritarian Complies

COMPLY
No data is transferred to any third party. All data generated by the extension — browsing activity, settings, session stats — is stored exclusively in Chrome's local storage on your device. No data is sent to any external server, database, or service.
COMPLY
Data is used only for core functionality. Website visit time is used solely to enforce your configured daily limits and display usage records. It is not cross-referenced, aggregated across users, or used for any secondary purpose.
COMPLY
No advertising use. The extension contains no advertising networks, tracking pixels, or analytics libraries. No user data is used to serve, target, or measure advertising.
COMPLY
No profiling or creditworthiness use. User data is never used to build behavioural profiles, assess creditworthiness, or support any financial determination.
COMPLY
Human-readable access only. Browsing activity data is only accessible to the user themselves, displayed in the extension's Records tab. It is never read programmatically by any party other than the extension's own code running locally on your device.

Scope of Data Access

The extension reads the hostname of your active browser tab (e.g. "youtube.com") every 15 seconds to accumulate a time total for that domain. Full URLs, search queries, page content, form data, and cookies are never read, stored, or transmitted. The extension has no capability to do so.

Questions

If you have questions about data use or this policy, contact the developer through the Chrome Web Store listing page.
